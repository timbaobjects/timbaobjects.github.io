Apollo

TimbaObject built an SMS based election monitoring system called Apollo. This tool enables observers send reports from the polling units during an election using text messages in a specific format. Messages sent are stored in a database, that is monitored on the Apollo web application platform. Analysis is performed on the received data for further use. Apollo has been used for election monitoring in countries like Nigeria, Zimbabwe, Tunisia, Zambia, and Azerbaijan.
